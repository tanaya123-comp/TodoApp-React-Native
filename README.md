# TodoApp-React-Native
- This is a Todo App built in React Native.
- This is a part of React Native Playlist of [The Net Ninja Channel](https://www.youtube.com/channel/UCW5YeuERMmlnqo4oq8vwUpg) on Youtube.
- Playlist Link - [React Native](https://www.youtube.com/watch?v=ur6I5m2nTvk&list=PL4cUxeGkcC9ixPU-QkScoRBVxtPPzVjrQ)

## Features
- Add Todo
- Delete Todo
- Custom checks on addition Of todos

## Features To Be Added
- Update Todo 
- Add Database for storage

## Installation Steps
- Clone Project
- Navigate to the Project Directory `cd TodoApp-React-Native`
- Install all Dependencies `npm install`
- To Run Project `npm start`



## Options for Running the Project
- Run on Android device/emulator
  -  Download the Expo App
  -  Scan the QR code above with the Expo app (Android) or the Camera app iOS
- Run on iOS simulator
